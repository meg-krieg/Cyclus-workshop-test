# Workshop Codespace
This repository holds GitHub Codespace configuration data and information for 
spinning up virtual `Cyclus` environments in the context of a live Cyclus 
Workshop setting or even a self-guided tutorial. 

## main 
The `main` branch holds a `.devcontainer` for setting up GitHub Codespace 
environment for new users via the web as well as examples, schema, and scripts 
helpful for running `Cyclus` demonstration simulations and and exploring 
outputs.


## gh-pages
The `gh-pages` branch holds a jekyll-based website solely for the organization 
of live workshops. It is built using GitHub Actions described in the 
`.github/workflows`. From this site, one can point to documentation on the main 
[fuelcycle.org](https://fuelcycle.org) site.
